# Review Memory

Use this file for recurring, reusable review lessons for this repository.

## What Belongs Here

- repeatable regression patterns discovered in review
- recurring validation gaps worth checking on future tasks
- stable standards-compliance pitfalls that frequently surface late

## What Does Not Belong Here

- findings for a single current review
- unresolved suspicions
- raw command output
- task-specific notes that belong in the handoff or task brief

## Entry Format

### <short title>
- Context: <when this matters>
- Lesson: <reusable guidance>
- Evidence: <verified fix, repeated pattern, or settled convention>
- Affected surfaces: <paths or domains>
